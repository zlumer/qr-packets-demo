offline mode
"cold storage"

This is the cold storage pinnacle of the current generation of mobile wallets. Go into airplane mode or use our built-in "in-app offline mode" and get all the benefits of hardware wallets with the usability of a mobile one. Transaction are confirmed through the Web interface and use of QR-codes.

online mode

To easy the use and attract more users we have a fully operational secure online wallet to compliment the offline version.

It can be used at almost the same security level as the offline mode. Use QR-codes to sign transactions and additional security such as face or finger print recognition.

transaction
confirmation

ICE WALLET will support DAPPs, Exchnages and Merchant Payments. Use it online or offline with the same 2-click secure authentication.
